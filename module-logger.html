Module: logger

Utility module for creating loggers with a globally equal, customised format.

(static) correlate(fn) → {Any}

Call the given function and add a correlation id to all messages logged during the function's execution unique to this correlate() call when using a logger created through module:logger.createLogger.
Parameters:
Name Type Description
fn function The function to call.
Source:
  • logger.js, line 94
Returns:
Return value of the given function, if any.
Type
Any
Example
const logger = require('./logger')
const log = logger.createLogger(__filename)
logger.correlate(() => {
  log.info('Test 1')
  setTimeout(() => { log.info('Test 2') }, 1000)
})
log.info('Test 3')
// Output (indented for better readability):
// {
//   "message": "Test 1",

//   "filename": "test.js",
//   "timestamp": "2018-08-12T19:27:09.381Z"
// }
// {
//   "message": "Test 3",
//   "level": "info",
//   "filename": "test.js",
//   "timestamp": "2018-08-12T19:27:09.385Z"
// }
// ...after 1 second:
// {

(static) createLogger(filename) → {winston.Logger}

Create a logger for the given module. Prints JSON messages to the console, adding information like the logging module's filename, a timestamp and a correlation id if called from within module:logger.correlate.
Parameters:
Name Type Description
filename string The filename to use then logging messages. Only the basename will be logged.
